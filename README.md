# 用于澳门城市大学的 $\LaTeX$ 模板



## Homework 说明

* Homework templates分为中文版与英文版，其区别在于是否使用了`ctex`  
* `main.pdf` 中可以预览模板效果
* `main.tex` 中进行正文编写
* `CityUhomework.sty`中编辑标题、页眉、学院、专业、作者等信息
* `reference.bib`中是参考文献列表
* `figures` 此文件夹存放所需的图片

## Pre 说明

* Beamer templates分为三种样式
* `main/slides.tex`中进行编写
* `main/slides.pdf`中进行预览
* `reference.bib`中是参考文献列表
* `figures` 此文件夹存放所需的图片
